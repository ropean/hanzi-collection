# 汉字集 - 界面设计规范

> 一个全功能的汉字学习与中华传统文化平台

---

## 一、产品定位

### 目标用户
- 五笔输入法学习者
- 中小学生及家长（识字、写字）
- 古典文学爱好者
- 对外汉语学习者
- 书法练习者

### 核心价值
**一站式汉字与传统文化学习平台**：查字、写字、识字、赏诗、品词、学成语，全面覆盖。

## 三、设计语言

### 3.1 设计原则

| 原则 | 说明 |
|------|------|
| **简洁克制** | 减少视觉噪音，突出内容本身 |
| **留白呼吸** | 充足间距，阅读舒适 |
| **层级分明** | 信息层次清晰，主次有序 |
| **文化意蕴** | 融入中式美学元素，但不堆砌 |
| **一致统一** | 组件风格、交互反馈保持一致 |

- 高度：64px
- 背景：白色 + 底部 1px 边框
- Logo：左侧，点击回首页
- 主导航：居中或靠左
- 工具栏：右侧（搜索、主题切换、收藏入口）


- 顶部栏高度：56px

## 八、动效规范

### 8.1 过渡动画

```css
/* 标准过渡 */
.transition-base {
  transition: all 0.2s ease-out;
}

/* 页面切换 */
.page-enter {
  opacity: 0;
  transform: translateY(10px);
}
.page-enter-active {
  opacity: 1;
  transform: translateY(0);
  transition: all 0.3s ease-out;
}

/* 卡片悬浮 */
.card:hover {
  transform: translateY(-2px);
  box-shadow: var(--shadow-lg);
}
```

### 8.2 笔顺动画

- 书写速度：可调节（慢/中/快）
- 笔画间隔：200-400ms
- 完成后停顿：500ms
- 支持暂停/继续

### 8.3 微交互

| 元素 | 交互 | 效果 |
|------|------|------|
| 按钮 | 点击 | 轻微缩放（0.98） |
| 卡片 | 悬浮 | 上浮 + 阴影增强 |
| 收藏按钮 | 点击 | 心跳动画 |
| 标签 | 选中 | 背景色渐变 |

## 九、无障碍设计

### 9.1 基本要求

- 所有交互元素可键盘访问
- 图片提供 alt 文本
- 表单元素有对应 label
- 颜色对比度 ≥ 4.5:1

### 9.2 ARIA 标签

```html
<!-- 搜索框 -->
<input 
  type="text" 
  role="searchbox"
  aria-label="搜索汉字、诗词、成语"
>

<!-- 笔顺播放按钮 -->
<button aria-label="播放汉字笔顺动画">
  ▶ 播放
</button>

<!-- 收藏按钮 -->
<button 
  aria-label="收藏此诗词"
  aria-pressed="false"
>
  ☆
</button>
```
